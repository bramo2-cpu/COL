Git Workflow for Task App Project

1. Open VS Code and navigate to project folder: C:\Apache24\htdocs\COL
2. Initialize Git repository (if not already done):
   git init
3. Check status of files:
   git status
4. Stage all files for commit:
   git add .
5. Commit changes with a descriptive message:
   git commit -m "Complete signup system with email, user list, and fixes"
6. Push changes to remote repository:
   git push origin main
7. Verify changes on GitHub/GitLab by visiting your repo online.
